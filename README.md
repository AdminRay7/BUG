## 🎯 About The Bot

**Ryan MD Bot** is a feature-rich WhatsApp Multi-Device bot built with Node.js. It includes a web dashboard, database management, and various moderation tools for WhatsApp groups.

**Hosted on:** [Bot-Hosting.net](https://bot-hosting.net)  
**Server Address:** `fi3.bot-hosting.net:23159`  
**Dashboard Port:** `3000`

## ✨ Features

### 🤖 Bot Features
| Feature | Description |
|---------|-------------|
| Multi-Device Support | Works with WhatsApp's latest multi-device beta |
| Pairing Code | Connect using 8-digit code (no QR scan needed) |
| Web Dashboard | Real-time bot monitoring and control |
| Database System | JSON-based file database for settings |

### 🛡️ Moderation Features
- ✅ Anti-link protection
- ✅ Anti-bad word filter  
- ✅ Anti-tag system
- ✅ Warning system
- ✅ Ban/Unban users
- ✅ Mute system
- ✅ Anti-delete messages

### 📝 Group Management
- ⭐ Welcome messages
- 👋 Goodbye messages
- 💬 Chatbot integration
- 📊 Message statistics
- 🔄 Auto-status updates

### 🌐 Web Dashboard
- Live bot status monitoring
- Group settings management
- User activity tracking
- Top users leaderboard
- Real-time message stats
